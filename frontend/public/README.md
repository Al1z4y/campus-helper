# Logo Instructions

To add your custom logo to Campus Helper:

1. Place your logo image file in this `public` folder
2. Name it `logo.png` (or update the path in the Logo component)
3. Recommended size: 32x32px to 128x128px
4. Supported formats: PNG, SVG, JPG

The logo will automatically appear in:
- The Sidebar header
- The Navbar (both logged-in and logged-out states)

If the logo image is not found, the app will automatically fall back to:
- A graduation cap icon in the Navbar
- "CH" text in the Sidebar
